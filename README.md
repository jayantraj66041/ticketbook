Project Setup
---------------------

Step 1:
```
git clone git@github.com:jayantraj66041/ticketbook.git
```

Step 2:
```
cd .\ticketbook\
```

Step 3:
```
pip install -r .\requirements.txt
```

Step 4:
```
python3 manage.py runserver 8000
```
